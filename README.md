## how to run
1. Get Public Key
https://dashboard.vapi.ai/org/api-keys  
2. install dependencies
```shell
cd vapi-vite
pnpm install
```
3.  create `.env` file and set the public key
``` shell
echo "VITE_VAPI_PUBLIC_KEY=" > .env
```
4. run the app
```shell
pnpm run dev
```
